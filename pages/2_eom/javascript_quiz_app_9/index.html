Акт ультразвукового измерения

Ознакомьтесь с актом ультразвукового измерения толщины стенок трубопровода горячего пара.

Для определения остаточной толщины стенки трубопровода применялся ультразвуковой толщиномер фирмы Panametrics модель 26MG. Данный толщиномер разработан для осуществления точных измерений толщины изделий, подверженных внутренней коррозии и эрозии. Он весит всего 240 г. С помощью его можно производить измерения при одностороннем доступе к объекту контроля.

Замер точек на окружности производился по часовой стрелке по ходу продукта. Для определения коррозионного поражения по нижней образующей трубы, измерения проводились по дуге 30 град (в местах сканирования указана минимальная остаточная толщина) в нижней части трубы.
